#Instalar Python en Windows
Ingresar a python.org
Click en Download
Verificar la versión adecuada, en este caso Python 3 en adelante
En caso de que ya esté instalado, solicitará actualizarse
En caso de instalarse de cero, marcar la casilla de "Agregar al path"

En CMD, escribir "python" para entrar al REPL
CTRL + Z + ENTER para salir

#En Linux
Abrir la temrinal
Escribir python3
CTRL + D para salir del REPL

¿Qué es el REPL?
Read, Evalue, Print & Loop (Lee, Evalúa, Imprime y Repite)
Esto se comunica con el interpré
